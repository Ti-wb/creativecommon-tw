Ubuntu Free Culture Showcase

2008-08-08

想讓作品揚名國際的機會又來了一個。Ubuntu 為下一個版本(應該是 8.10，也就是今年十月會推出的版本)舉辦了「Free Culture Showcase」活動，一旦入選、你的作品就可能會「灑」到全球數百萬台電腦上 — 這比台灣現在很多藝人賣得都多啦！以下引自 Ubuntu Wiki，並附上簡譯：

For a long time now, Ubuntu has included on the CD a small collection of media files that can be used to show off a new Ubuntu system. This content has always included a sound and a video clip, both using the free Ogg Vorbis and Theora formats.

創用 CC 授權解說手冊開放下載

2008-04-10

台灣創用 CC 計畫的兩本說明手冊相繼面世了。去年底 (2007/11) 由創用 CC、教育部與台大國發所劉靜怡教授合作的《創作分享˙快樂使用—簡介創用CC授權》率先印製完畢；今年三月底由創用 CC 與文建會、台大法學院謝銘洋教授合作的《創用CC授權指引--政府部門與民間組織篇》也隨之出爐。兩本手冊的電子檔 (PDF) 都採用姓名標示－相同方式分享授權。

《創作分享˙快樂使用—簡介創用CC授權》主要針對教育領域的使用情境設計規劃，以深入淺出的筆法說明創用 CC 授權的核心價值、四大要素以及六大授權條款，並且蒐羅整理推廣期間教師同學們的提問與回應。

著重在機關組織的《創用CC授權指引—政府部門與民間組織篇》則討論此類型的著作匯集單位，欲將其擁有之著作釋出公眾利用時，應該採取之步驟以及關於釋出之評估考量。兩本手冊中都收錄了一些國內外使用創用 CC 的優秀範例及常見問答集。歡迎多多利用。

除了 PDF 版之外，為了方便取用，創用 CC 也將於稍後推出 HTML 圖文版，敬請期待。
